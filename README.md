# GQL Mock project

## What is this?
It's a demo for creating an AWS AppSync API with a few Lambda functions!

## Stuff you need locally

yarn:
```
brew install yarn
```

serverless:
```
npm i -G serverless
```

AWS setup:
https://serverless.com/framework/docs/providers/aws/guide/quick-start/

## How do I use this?

From within a checkout of this code...

```
yarn
npm run deploy
```
